# Contact-Relative Invariant EKF — Project Page

Proprioceptive state estimation for wide-footed wearable robots: a contact-synthesized
relative-SLAM correction that rescues the contact-aided InEKF from wide-foot divergence
(standard ~14 m → contact-relative ~6.6 m final position error, no camera).

**Live page:** https://sangpaa.github.io/contact-relative-inekf/

Static single-page site (`index.html` + `static/`). No build step.
